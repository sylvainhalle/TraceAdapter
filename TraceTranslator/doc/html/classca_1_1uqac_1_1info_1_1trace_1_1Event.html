boolean ca.uqac.info.trace.Event.isFlat ( )

Determines if an event is flat. An event is considered flat if it contains only top-level parameters; for example: 

 <Event>
   <a>123</a>
   <b>345</b>
 </Event>

is flat while 

 <Event>
   <a>123</a>
   <b>
     <c>222</c>
   </b>
 </Event>

is not.
