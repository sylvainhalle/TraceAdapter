Detailed Description

Parses a trace written as a comma-separated values (CSV) file. A CSV file typically has the form: 

 pname1,pname2,...,pnamen
 val1,val2,...,valn
 val1,val2,...,valn
 ...

Where the pnames are the names for each field (column), and the remaining lines give comma-separated list of values for each parameter. The parser is robust, in that it ignores lines made of whitespace only, and accepts the "#" character for comment lines
