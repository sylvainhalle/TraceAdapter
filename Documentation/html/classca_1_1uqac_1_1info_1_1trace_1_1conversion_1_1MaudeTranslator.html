Detailed Description

Translates an input trace and a formula for the Maude rewriting engine. The translation follows the syntax described in the following paper:

  • Klaus Havelund, Grigore Roşu. (2001). <cite>Monitoring Programs using Rewriting.</cite> Automated Software Engineering 2001 (ASE'01).
Author:
Aouatef Mrad
