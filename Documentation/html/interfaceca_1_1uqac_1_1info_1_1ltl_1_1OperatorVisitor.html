Detailed Description

Implementation of the visitor pattern to traverse LTL syntax trees. Usage for a visitor:

  1. Instantiate a visitor v
  2. Feed it to some operator op by calling op.accept(v)
  3. Internally, the operator will first call accept on its operand(s), and then call v.visit(this)

The end result is that the visitor's accept method will be called exactly once for each element of the syntax tree, in a sequence corresponding to a postfix traversal.
