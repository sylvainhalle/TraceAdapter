Detailed Description

Representation of a finite-state machine edge between two states. A triplet contains:

  1. The source state label
  2. The edge label
  3. The destination state label

A set of such triplets defines a finite-state machine.

Builds a triplet from a structured string. The string format is: 

 s_label e_label d_label

where each element represents the source, edge and destination label, respectively, separated by any number of whitespace.
