void ca.uqac.info.trace.generation.FsmGenerator.parseFromString ( String  fsmString) [protected]

Defines the finite-state machine from a structured string. The string itself is just a succession of triplets separated by spaces. See FsmGenerator.FsmTriplet(String) for a definition of each triplet's format.

By definition, the FSM's initial state label is the "from" label of the first triplet. It may occur in more than one triplet (that is, the FSM need not be deterministic).

Parameters:
fsmStringThe structured string
