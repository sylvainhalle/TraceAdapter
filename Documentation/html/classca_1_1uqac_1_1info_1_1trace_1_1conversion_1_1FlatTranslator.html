boolean ca.uqac.info.trace.conversion.FlatTranslator.isEquivalencePreserved ( )

Determines if the equivalence has been preserved during the translation. The equivalence is not preserved when there is more than one occurrence of a parameter inside an event, as only one of these values will survive in the translated trace.

Returns:
True if the equivalence was preserved, false otherwise
