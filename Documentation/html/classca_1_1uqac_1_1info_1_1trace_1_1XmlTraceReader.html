Detailed Description

Parses a trace written as an XML file file. The XML file is expected to have the form: 

 <Trace>
   <Event>
   ...
   </Event>
   <Event>
   ...
   </Event>
   ...
 </Trace>

where each Event element itself nests elements that are taken as the parameters for that event. The parser uses Java's own DOM parser and hence should accept any well-formed XML file.
