Detailed Description

Takes care of the execution of various tools in a uniform way. The intended workflow is as follows:

  1. Instantiate a child of Execution (e.g. SaxonExecution)
  2. Feed the instance the trace to monitor using setTrace
  3. Feed the instance the property to monitor using setProperty (if required)
  4. Feed the instance the signature to use using setSignature (if required)
  5. Launch the tool using run
  6. Collect the outcome, time and memory consumed using getReturnValue, getTime, getMemory
