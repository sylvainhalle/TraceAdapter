Detailed Description

Generates a trace of events based on the definition of a Petri net. For more information about Petri nets, see Wikipedia's entry.

See parseFromString for a precise definition of the input format to specify the Petri net.

void ca.uqac.info.trace.generation.PetriNetGenerator.parseFromString ( String  s) throws java.text.ParseException [protected]

Defines the Petri net from a structured string. The string itself is just a succession of "words" separated by whitespace with the following structure:

T label1 label2
Defines an arrow from a transition named label1 to a place named label2
P label1 label2
Defines an arrow from a place named label1 to a transition named label2
M label1 x
Defines an initial marking by putting x tokens into a place named label1

For example, Petri net N shown on Wikipedia's entry would be represented as: 

 T T1 p1
 P p1 T2
 T T2 p2
 P p2 T3
 M p1 0
 M p2 0

Carriage returns are optional, the whole text could have been put on a single line. The order in which lines are input does not matter: transitions and places are created the first time their label is seen by the parser.

Caveat emptor: the parser assumes the string is well-formed but does not check it. A malformed string will likely cause a runtime error of some kind.

Parameters:
sThe structured string
