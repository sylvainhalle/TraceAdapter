Detailed Description

The TraceGenerator defines the basic methods one can use to produce event traces of various kinds. It also provides functionality to read command-line arguments, produce an output trace and initialize a random number generator.

Generate an event trace. There are no basic assumptions on that trace: it may be the same on every call to generate, or different. It is up to each trace generator to determine the appropriate behaviour for this method.
