Detailed Description

An event is a data structure representing the occurrence of some action, command, message or request in a system under observation. Formally, it is represented as a nested set of parameter-value pairs.

A natural data format compatible with this definition is XML. As a matter of fact, internally an event simply contains a DOM org.w3c.xml.Node, and serves as an interface between that node's content and the outside world.

boolean ca.uqac.info.trace.Event.isMultiValued ( )

Determines if an event is multi-valued. This is the case exactly when there exists more than one instance of some parameter carrying a different value. For example: 

 <Event>
   <a>123</a>
   <a>456</a>
   <b>345</b>
 </Event>

is multi-valued while 

 <Event>
   <a>123</a>
   <b>
     <c>222</c>
   </b>
 </Event>

is not.

Returns:
True if event is multi-valued, false otherwise
